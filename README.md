# RGCacheKit

Coredata helper.

## Installation

### Cocoapods
To integrate RGCacheKit into your Xcode project using CocoaPods, specify it in your Podfile:

#### Swift 4.0
```
  pod 'RGCacheKit', :git => 'https://github.com/riteshhgupta/RGCacheKit.git'
```
